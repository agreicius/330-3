Theorem 1.13.3. Separable minimal polynomial.

Let \(K/F\) be a field extension. An element \(\alpha\in K\) is separable over \(F\) if and only if its minimal polynomial is separable over \(F\text{.}\) Using logical shorthand:
\begin{equation*} \alpha \text{ separable} \iff m_{\alpha, F}(x) \text{ separable}\text{.} \end{equation*}

Proof.

The reverse direction is clear: if \(m_{\alpha,F}(x)\) is separable, then \(\alpha\) is separable, since \(m_{\alpha,F}(\alpha)=0\text{.}\)
For the forward implication, assume \(\alpha\) is separable over \(F\text{,}\) in which it is the root of a separable polynomial \(g(x)\in F[x]\text{.}\) It follows from CorollaryÂ 1.5.3 that \(m_{\alpha,F}(x)\mid g(x)\text{.}\) But then \(m_{\alpha,F}(x)\) must be separable. Indeed, \(m_{\alpha,F}(x)\) cannot have a repeated root: if \((x-\beta)^2\mid m_{\alpha,F}(x)\) for some \(\beta\in \overline{F}\text{,}\) then \((x-\alpha)^2\mid g(x)\text{,}\) which is impossible since \(g\) is separable.
