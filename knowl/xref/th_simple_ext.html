Theorem 1.4.12. Simple extensions.

Let \(K/F\) be a field extension and let \(\alpha\in K\text{.}\)
  1. The following statements are equivalent.
    1. \(\alpha\) is algebraic.
    2. \(F[\alpha]=F(\alpha)\cong F[x]/(f)\) for some irreducible polynomial \(f(x)\in F[x]\text{.}\)
    3. \(F(\alpha)/F\) is finite: i.e., \([F(\alpha)\colon F]< \infty\text{.}\)
    Furthermore, when \(\alpha\) is algebraic, there is a basis of \(F(\alpha)\) of the form \(B=\{1,\alpha, \alpha^2,\dots, \alpha^{n-1}\}\text{.}\)
  2. The following statements are equivalent.
    1. \(\alpha\) is transcendental.
    2. \(F[\alpha]\cong F[x]\) and \(F(\alpha)\cong F(x)\text{.}\)
    3. \(F(\alpha)/F\) is infinite: i.e., \([F(\alpha)\colon F]=\infty\text{.}\)

Proof.

The key to proving both statements of the theorem is to understand the “evaluation at \(\alpha\)” ring homomorphism
\begin{align*} \phi \colon F[x] \amp \rightarrow K\\ f(x) \amp \mapsto f(\alpha)\text{.} \end{align*}
Note first that we have
\begin{equation*} \phi(F[x])\cong F[x]/\ker \phi\text{,} \end{equation*}
which implies \(\ker \phi\) is a prime ideal of \(F[x]\) (since \(\phi(F[x])\leq K\) is an integral domain). It follows that \(\ker\phi=(0)\) or \(\ker \phi=(f)\) for some irreducible polynomial \(f\in F[x]\text{.}\) Furthermore, since
\begin{equation*} \ker\phi=\{f\in F[x]\colon f(\alpha)=0\}\text{.} \end{equation*}
we see immediately that the element \(\alpha\) is transcendental if and only if \(\ker\phi=(0)\) if and only if \(\phi\) is injective. Similarly, \(\alpha\) is algebraic if and only if \(\ker \phi=(f)\) for some irreducible polynomial \(f\text{.}\)
Lastly, observe that
\begin{equation*} \phi(F[x])=\{\phi(f)\colon f\in F[x]\}=\{f(\alpha)\colon f(x)\in F[x]\}=F[\alpha]\text{,} \end{equation*}
the subring of \(K\) generated over \(F\) by \(\alpha\text{.}\)
Having given a rather thorough analysis of \(\phi\) we are ready to prove statements (1) and (2).

Proof of (1).

We have
\begin{align*} \alpha \text{ algebraic} \amp \iff \phi(F[x])\cong F[x]/(f) \text{ for some irreducible } f\\ \amp \iff F[\alpha]=\phi(F[x])\cong F[x]/(f) \text{ for some irreducible } f\\ \amp \iff F[\alpha]=F(\alpha)\cong F[x]/(f) \text{ for some irreducible } f\text{,} \end{align*}
where the last equivalence uses the fact that \(F[\alpha]=F(\alpha)\) if and only if \(F[\alpha]\) is a field. (Theorem 1.4.4) This proves (i)\(\iff\)(ii).
Next, (ii)\(\implies\)(iii) since if \(f\) is irreducible, the quotient \(F[x]/(f)\) is a finite field extension of \(F\) of degree \(n=\deg f\text{.}\)
Lastly (iii)\(\implies\)(i) since if \(\dim_F F(\alpha)=n\text{,}\) then the set \(\{\alpha^n, \alpha^{n-1},\dots, \alpha, 1\}\) must be linearly dependent, in which case there is a nontrivial linear combination
\begin{equation*} a_n\alpha^{n}+a_{n-1}\alpha^{n-1}+\cdots +a_1\alpha+a_0=0\text{.} \end{equation*}
But then we have \(f(\alpha)=0\text{,}\) where \(f\) is the nonzero polynomial \(f(x)=\anpoly\text{.}\) Thus \(\alpha\) is algebraic, as desired.

Proof of (2).

First observe that the equivalence (i)\(\iff\)(iii) follows from (1) above simply by taking negations. Furthermore, we have (ii)\(\implies\)(iii) since \([F(x)\colon F]=\infty\text{,}\) as we saw in Example 1.4.3. Thus it remains only to show (i)\(\implies\)(ii).
Assume \(\alpha\) is transcendental. We saw above that this is equivalent to our evaluation map \(\phi\) being injective. Thus we have \(F[\alpha]=\phi(F[x])\cong F[x]\text{.}\) Since \(F(x)=\Frac F[x]\text{,}\) we may use Theorem 1.1.9 to get an injective map
\begin{align*} \widetilde{\phi}\colon F(x) \amp \rightarrow K\\ f(x)/g(x) \amp \mapsto f(\alpha)/g(\alpha)\text{.} \end{align*}
Thus \(\widetilde{\phi}(F(x))\) is a subfield of \(K\) containing \(F\) and \(\alpha\text{.}\) Since in turn any field containing \(F\) and \(\alpha\) must also contain all elements of the form \(f(\alpha)/g(\alpha)\text{,}\) where \(f\) and \(g\) are polynomials, we see that \(\widetilde{\phi}(F(x))\) in fact the smallest such field: i.e., we have \(F(\alpha)=\widetilde{\phi}(F(x))\cong F(x)\text{,}\) as desired.
