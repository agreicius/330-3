Remark 1.20.12. Discriminant and symmetric polynomials.

The expression (1.36) defining the discriminant is symmetric in the roots \(\alpha_i\text{.}\) More precisely, and more generally, a polynomial \(f(x_1,x_2,\dots, x_n)\in F[x_1,x_2,\dots, x_n]\) in the variables \(x_1,x_2,\dots, x_n\) is symmetric if for all \(\sigma\in S_n\) we have
\begin{equation*} f(x_{\sigma(1)},x_{\sigma(2)},\dots, x_{\sigma(n)})=f(x_1,x_2,\dots, x_n)\text{.} \end{equation*}
There is a rich, wonderful theory of symmetric polynomials that we do not have time to go into. However, it is worth pointing out the so-called elementary symmetric polynomials, a list of \(n\) specific symmetric polynomials \(s_k(x_1,x_2,\dots, x_n)\text{,}\) \(1\leq k\leq n\text{.}\) These are defined in the following manner. Consider the polynomial \(G(x,x_1,x_2,\dots, x_n)\) in the \(n+1\) variables \(x,x_1,\dots, x_n\) defined as
\begin{equation} G(x,x_1,\dots, x_n)=(x-x_1)(x-x_2)\cdots (x-x_n)\text{.}\tag{1.37} \end{equation}
Expanding \(G\) out as a polynomial in \(x\) with coefficients in \(F[x_1,x_2,\dots, x_n]\text{,}\) we have
\begin{align*} G \amp = x^n-(x_1+x_2+\cdots +x_n)x^{n-1}+\cdots+(-1)^n(x_1x_2\cdots x_n)\\ \amp = x^n-s_1(x_1,x_2,\dots, x_n)x^{n-1}+s_2(x_1,x_2,\dots, x_n)x^{n-2}+\cdots +(-1)^{n-1}s_{n-1}(x_1,x_2,\dots, x_n)+(-1)^{n}s_n(x_1,x_2,\dots, x_n)\\ \amp = x^n+\sum_{k=1}^n(-1)^ks_k(x_1,x_2,\dots, x_n)x^k\text{.} \end{align*}
Each polynomial \(s_k(x_1,x_2,\dots, x_n)\) is a symmetric polynomial in \(x_1,x_2,\dots, x_n\text{,}\) and we call these the elementary symmetric polynomials for \(n\text{.}\) When \(n=3\text{,}\) for example, we have
\begin{align*} s_1 \amp =x_1+x_2+x_3 \amp s_2\amp = x_1x_2+x_1x_3+x_2x_3 \amp s_3\amp=x_1x_2x_3 \text{.} \end{align*}
Amazingly, any symmetric polynomial can be written as a polynomial expression of the elementary polynomials \(s_k\text{.}\) In other words, if \(g(x_1,x_2,\dots, x_n)\) is a symmetric polynomial, then we have
\begin{equation*} g=H(s_1,s_2,\dots, s_n) \end{equation*}
for some polynomial \(H(x_1,x_2,\dots, x_n)\text{.}\)
Returning at last to the discriminant, with the setup as in DefinitionÂ 1.20.4, we have
\begin{equation*} \Delta_f=g(\alpha_1,\alpha_2,\dots, \alpha_n)\text{,} \end{equation*}
where \(g\) is the symmetric polynomial
\begin{equation*} g(x_1,x_2,\dots, x_n)=\prod_{1\leq i< j\leq n}(x_j-x_i)^2\text{.} \end{equation*}
By above we have \(g=H(s_1,s_2,\dots, s_n)\) for some polynomial \(H\text{,}\) in which case
\begin{align*} \Delta_f \amp= g(\alpha_1,\dots, \alpha_n)\\ \amp = H(s_1(\alpha_1,\dots, \alpha_n),\dots, s_n(\alpha_1,\dots, \alpha_n))\text{.} \end{align*}
Furthermore, since \(f(x)=G(x,\alpha_1,\alpha_2,\dots, \alpha_n)\text{,}\) where \(G\) is as in (1.37), we see that by definition
\begin{align*} s_k(\alpha_1,\alpha_2,\dots, \alpha_n) \amp = (-1)^ka_{n-k}\text{.} \end{align*}
Thus we have
\begin{align*} \Delta_f\amp=H(s_1(\alpha_1,\dots, \alpha_n),\dots, s_n(\alpha_1,\dots, \alpha_n))\\ \amp=H(-a_{n-1},a_{n-2},-a_{n-3}, \dots, (-1)^n\, a_0)\text{,} \end{align*}
from whence it follows that \(\Delta_f\) is a polynomial expression in the coefficients \(a_k\) of \(f\text{.}\)
