Definition 1.1.2. Units of a ring.

Let \(R\) be a ring. An element \(r\in R\) is a unit (or invertible) if there is an element \(s\in R\) satisfying \(rs=sr=1\text{.}\) The set of all units of \(R\text{,}\) denoted \(R^*\text{,}\) is a group under the ring multiplication operation and is called the group of units of \(R\text{.}\)
