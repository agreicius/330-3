Definition 1.7.1. Split completely.

Let \(K/F\) be a field extension. A polynomial \(f\in F[x]\) splits completely over \(K\) if \(f\) factors over \(K\) as
\begin{equation} f(x)=c(x-\alpha_1)(x-\alpha_2)\cdots (x-\alpha_n)\text{,}\tag{1.10} \end{equation}
for some integer \(n\geq 0\text{,}\) constant \(c\in F\text{,}\) and elements \(\alpha_i\in K\text{.}\) We say that the field \(K\) splits the polynomial \(f\) in this case.
More generally, if \(\mathcal{P}\subseteq F[x]\) is a set of polynomials, we say that \(K\) splits \(\mathcal{P}\) if it splits each polynomial \(f\in \mathcal{P}\text{.}\)
