Corollary 1.17.3. Simple subfields of Galois extensions.

Let \(L/F\) be a finite Galois extension with Galois group \(G=\Gal(L/F)\text{,}\) let \(\alpha\) be an element of \(L\text{,}\) and let \(O_{\alpha}=\{\sigma(\alpha)\colon \sigma\in G\}\) be the set of Galois conjugates of \(\alpha\text{.}\)
  1. \(\displaystyle \Aut_{F(\alpha)}(L)=\Stab_G(\alpha)\)
  2. \(m_{\alpha,F}(x)=\prod_{\beta\in O_\alpha}(x-\beta) \text{.}\)
  3. \([F(\alpha)\colon F]=\deg_F \alpha=\abs{O_{\alpha}}=[G\colon O_{\alpha}] \text{.}\)
  4. Given an intermediate field \(K\text{,}\) we have \(\alpha\in K\) if and only if \(\sigma(\alpha)=\alpha\) for all \(\sigma\in \Aut_{K}(L)\text{.}\)
    In particular, \(\alpha\in F\) if and only if \(\sigma(\alpha)=\alpha\) for all \(\sigma\in G\text{.}\)
  5. Given an intermediate field \(K\) we have \(F(\alpha)=K\) if and only if \(\Aut_{K}(L)=\Stab_G(\alpha)\text{.}\)
    In particular, we have \(F(\alpha)=L\) if and only if \(\Stab_{G}(\alpha)=\{\id\}\text{.}\)
