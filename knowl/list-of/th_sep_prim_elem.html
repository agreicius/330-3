Theorem 1.13.9. Primitive element theorem.

Let \(K/F\) be a finite separable extension. There is an element \(\alpha\in K\) such that \(K=F(\alpha)\text{.}\) Using logical shorthand:
\begin{equation*} K/F \text{ finite separable} \implies K=F(\alpha) \text{ simple}\text{.} \end{equation*}
