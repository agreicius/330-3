Example 0.4.3. \(D_3\) is isomorphic to \(S_3\).

Use Theorem 0.4.2 to show that \(D_3\) is isomorphic to \(S_3\)
Solution.
Using cycle notation for \(S_3\text{,}\) it is easy to verify that the elements \(\rho=(123)\) and \(\sigma=(12)\) satisfy the following relations:
\begin{align*} \rho^3 \amp =\sigma^2=e \amp \sigma \rho=\rho^2\sigma\text{.} \end{align*}
According to Theorem 0.4.2 there is a homomorphism \(\phi\colon D_3\rightarrow S_3\) satisfying \(\phi(r)=\rho, \phi(s)=\sigma\text{,}\) and \(\im \phi=\angvec{\rho,\sigma}\leq S_3\text{.}\) (The full formula for \(\phi\) is \(\phi(r^js^k)=\rho^js^k\text{.}\)) Since \(\angvec{\rho,\sigma}=S_3\text{,}\) we see that \(\phi\) is surjective. Since \(\abs{D_3}=\abs{S_3}=6\text{,}\) we conclude that \(\phi\) is bijective, hence an isomorphism. This proves \(D_3\cong S_3\text{.}\)
