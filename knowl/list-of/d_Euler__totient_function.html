Definition 0.4.4. Euler totient function.

The Euler totient function is the function \(\phi\colon \Z_+\rightarrow \Z_+\) defined as
\begin{equation*} \phi(n)=\abs{\{k\in \{1,2,\dots, n\}\colon \gcd(k,n)=1\}}\text{.} \end{equation*}
In other words, \(\phi(n)\) is the number of integers \(k\) satsifying \(1\leq k\leq n\) satisfying \(\gcd(k,n)=1\text{.}\)
