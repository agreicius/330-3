Theorem 1.5.5. Tower law.

Let \(L/K/F\) be a tower of (possibly infinite) field extensions.
  1. If \(B_1\) is a basis of \(K\) and \(B_2\) is a basis of \(L\) as a \(K\)-vector space, then the set
    \begin{equation} B=\{\alpha\cdot \beta\colon \alpha\in B_1, \beta\in B_2\}\tag{1.5} \end{equation}
    is a basis of \(L\) as an \(F\)-vector space.
  2. We have
    \begin{equation} [L\colon F]=[L\colon K][K\colon F]\text{.}\tag{1.6} \end{equation}

Proof of (1).

Let \(B=\{\alpha\cdot \beta\colon \alpha\in \B_1, \beta\in B_2\}\text{,}\) as in the theorem statement. We will show directly that \(B\) is a basis of \(L\) as an \(F\)-vector space, by showing that it spans and is linearly independent.

\(B\) spans.

Given any \(\alpha\in L\text{,}\) since \(B_2\) is a \(K\)-basis of \(L\text{,}\) we can write
\begin{equation*} \alpha=c_1\beta_1+c_2\beta_2+\cdots +c_n\beta_n=\sum_{j=1}^nc_j\beta_j \end{equation*}
for some elements \(c_j\in K\text{.}\) Next since \(B_1\) is a basis of \(K\) over \(F\text{,}\) we can find a finite set \(\{\alpha_1,\alpha_2,\dots, \alpha_m\}\subseteq B_1\) such that for each \(1\leq j\leq n\) we have
\begin{equation*} c_j=d_{1j}\alpha_1+d_{2j}\alpha_2+\cdots +d_{mj}\alpha_m=\sum_{i=1}^md_{ij}\alpha_i \end{equation*}
for some elements \(d_{ij}\in F\text{.}\) Substituting these expressions for \(c_j\) into the first linear combination above, we conclude that
