Theorem 1.12.10. Separable irreducible.

Let \(F\) be a field. An irreducible polynomial \(f\in F[x]\) is separable if and only if \(f'\ne 0\text{.}\)

Proof.

By TheoremÂ 1.12.7, \(f\) is separable if and only if \((f,f')=1\text{;}\) so it suffices to show that for an irreducible polynomial, \((f,f')=1\) if and only if \(f'\ne 0\text{.}\)
Assume \(f=\anpoly\) is irreducible. Since \(f'(x)=na_nx^{n-1}+\cdots 2a_2x+a_1\text{,}\) we see that either \(\deg f'< \deg f\text{,}\) or \(f'=0\text{.}\) Since the only factors of \(f\text{,}\) up to multiplication by units, are \(f\) and \(1\text{,}\) we see that \((f,f')=1\) or \((f,f')=f\text{.}\) Since \(f'\ne 0\) implies \(\deg f'< f\text{,}\) we see that the second condition attains if and only if \(f'=0\text{.}\) Equivalently, \((f,f')=1\) if and only if \(f'=0\text{.}\)
