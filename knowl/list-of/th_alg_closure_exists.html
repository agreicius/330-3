Theorem 1.8.7. Algebraic closure exists.

Given any field \(F\text{,}\) there is an algebraic closure \(K/F\text{.}\)

Proof.

Fix a field \(F\text{.}\) Our proof will make use of the polynomial ring
\begin{equation*} R=F[\{ t_{f,j}\}] \end{equation*}
in infinitely many variables \(t_{f,j}\text{,}\) where \(f\in F[x]\) is a nonconstant polynomial, and \(1\leq j\leq \deg f\text{.}\) In other words, each pair \((f,j)\text{,}\) where \(f\) is a nonconstant polynomial and \(j\) is a positive integer bounded by \(\deg f\text{,}\) is associated with a unique variable \(t_{f,j}\text{.}\) Without getting too technical, the polynomial ring \(F[\{t_{f,j\}]\) is defined as the set of all finite polynomial combinations of the infinite variable set \(\{t_{f,j}\colon f\in F[x]-F, 1\leq j\leq \deg f\}\text{.}\)
Next we will define an ideal \(I\) of \(R\text{.}\) For each monic nonconstant polynomial \(f\in F[x]\text{,}\) let
\begin{align} g_f(x)\amp =f(x)-\prod_{j=1}^n(x-t_{f,j}) \tag{1.11}\\ \amp =g_{f,n}x^n+g_{f,n-1}x^{n-1}+\cdots +g_{f,1}x+g_{f,0}\tag{1.12}\\ \amp \in R[x]=F[\{t_{f,j}\}][x] \text{.}\tag{1.13} \end{align}
Observe that the coefficients \(g_{f,j}\) here are polynomials in the variables \(\{t_{f,1},t_{f,2},\dots ,t_{f,\deg f}\}\text{.}\) With this notation in place we define \(I\) to be the ideal generated by the polynomials \(g_{f,t}\in R\text{:}\) i.e.,
\begin{equation*} I=(\{g_{f,j}\colon f\in F[x]-F \text{ monic }, 1\leq j\leq \deg f\})\text{.} \end{equation*}
Having assembled all the necessary ingredients, the rest of the proof proceeds as follows:
  1. Show that \(I\ne R\text{,}\) and thus is contained in a maximal ideal \(\mathfrak{m}\leq R \text{.}\)
  2. Show that the field \(K=R/\mathfrak{m}\) is an algebraic extension of \(F\) that splits \(F[x]\text{.}\)
Once we have shown (2), we conclude from Corollary 1.8.6 that \(K/F\) is an algebraic closure.

Proof: \(I\ne R\).

Assume by contradiction that \(I=R\text{.}\) This would mean that we have
\begin{equation} 1=q_1g_{f_1,j_1}+q_2g_{f_2,j_2}+\cdots +q_mg_{f_m, j_m}\tag{1.14} \end{equation}
for some \(f_1,f_2,\dots, f_m\in F[x]-F\) and \(g_{f_k,j_k}\in I\text{.}\) By Corollary 1.7.9 there is a splitting field \(L/F\) of the \(f_k\text{.}\) For each \(1\leq k\leq m\text{,}\) pick a root \(\alpha_k\in L\) of \(f_k\) and define
\begin{equation*} \phi\colon R=F[\{t_{f,j}\}]\rightarrow L \end{equation*}
to be the evaluation map that for each \(f_k\) sends \(t_{f_k,j}\mapsto \alpha_{k,j}\) for each \(1\leq j\leq \deg f_k\text{,}\) where \(\alpha_{k,j}\) are the roots of \(f_k\text{;}\) and which sends all other variables \(t_{f,j}\) to \(0\text{.}\) We claim that
\begin{equation} \phi(g_{f_k, j_k})=0\tag{1.15} \end{equation}
for all \(k\text{.}\) Assuming this is so, we have reached a contradiction since then
\begin{equation*} \phi(1)=\phi(q_1)\phi(g_{f_1,j_1})+\cdots +\phi(q_m)\phi(g_{f_m,j_m})=0\text{.} \end{equation*}
To see why the claim ((1.15)) is true, we first extend the map \(\phi\) as usual to a map
\begin{equation*} \phi\colon R[x]\rightarrow L[x] \end{equation*}
by sending \(x\) to \(x\text{.}\) For each \(g_{f_k}\) as in ((1.12)), we have
\begin{align*} \phi(g_{f_k})\amp=f_k(x)-\prod_{j=1}^{\deg f_{k}}(x-\phi(t_{f_k,j})) \\ \amp=f_k(x)-\prod_{j=1}^{\deg f_{k}}(x-\alpha_{k,j})\\ \amp = 0\text{,} \end{align*}
since we have
\begin{equation*} f(x)=\prod_{j=1}^{\deg f_k}(x-\alpha_{k,j}) \end{equation*}
in \(L\text{.}\) On the other hand, we have
\begin{equation*} \phi(g_{f_k})=\phi(g_{f_k,n})x^n+\cdots +\phi(g_{f_k,1})x+\phi(g_{f_k,0})\text{.} \end{equation*}
For the polynomial on the right-hand side to be equal to 0 in \(L[x]\text{,}\) we must have the coefficients \(\phi(g_{f_k, j})=0\) for all \(j\text{,}\) as claimed.

\(K=R/\mathfrak{m}\) is algebraic closure.

Since \(I\ne R\text{,}\) there is a maximal ideal \(\mathfrak{m}\supseteq I\text{.}\) Let \(K=R/\mathfrak{m}\text{.}\) As usual, the resriction to \(F\) of the quotient map \(R\rightarrow R/\mathfrak{m}\) is injective, and we identify \(F\) with its isomorphic image in \(K=R/\mathfrak{m}\text{.}\)
First we show that \(K\) splits all nonconstant polynomials in \(F[x]\text{.}\) Clearly it suffices to show that \(K\) splits all nonconstant monic polynomials \(f\in F[x]\text{.}\) To see why this is true, first observe that the quotient map \(\pi\colon R\rightarrow R/\mathfrak{m}\) extends to a map of polynomial rings
\begin{equation*} \pi \colon R[x]\rightarrow R/\mathfrak{m}[x] \end{equation*}
sending \(x\) to \(x\text{.}\) For each monic nonconstant polynomial \(f\in F[x]\text{,}\) we have
\begin{align*} \pi(g_f) \amp =f(x)-\prod_{j=1}^{\deg f}(x-\overline{t_{f,j}})\\ \amp =\overline{g_{f,n}x^n+\cdots +\overline{g_{f,1}}x+\overline{g_{f,0}}\\ \amp = 0\text{,} \end{align*}
since by definition of \(I\) we have \(g_{f,j}\in I\leq \mathfrak{m}\text{.}\) It follows that
\begin{equation*} f(x)=\prod_{j=1}^{\deg f}(x-\overline{t_{f,j}})\in K[x]\text{,} \end{equation*}
showing that \(f(x)\) splits completely in \(K[x]\text{.}\)
