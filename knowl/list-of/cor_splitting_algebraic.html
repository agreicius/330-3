Corollary 1.7.7. Splitting fields are algebraic.

Let \(F\) be a field, let \(\mathcal{P}\subseteq F[x]\) be a set of polynomials. If \(K/F\) is a splitting field of \(\mathcal{P}\text{,}\) then \(K/F\) is algebraic.

Proof.

Clearly \(K\) splits \(\mathcal{P}\text{.}\) By Theorem 1.7.6, we have \(K=F(A)\text{,}\) where \(A\) is the set of all roots of all polynomials of \(\mathcal{P}\text{.}\) Since each element of \(A\) is algebraic over \(F\text{,}\) we see by Corollary 1.6.11, that \(K=F(A)\) is algebraic over \(F\text{.}\)
