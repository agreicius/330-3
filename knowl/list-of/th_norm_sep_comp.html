Theorem 1.15.3. Normal/separable revisited.

Let \(\overline{F}/F\) be an algebraic closure, and let \(K/F\) be a subextension.
  1. In general we have \(\Aut_F(K)\subseteq \Hom_F(K,\overline{F})\text{.}\)
  2. \(K/F\) is normal if and only if \(\Hom_F(K,\overline{F})=\Aut_F(K)\) if and only if \(m_{\alpha,F}(x)\) splits completely over \(K\) for all \(\alpha\in K\text{.}\)
  3. If \(K/F\) is finite, then \(K/F\) is separable if and only if
    \begin{equation} \abs{\Hom_F(K,\overline{F})}=[K\colon F]\text{.}\tag{1.26} \end{equation}
