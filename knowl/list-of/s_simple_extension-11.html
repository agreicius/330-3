Example 1.4.10. Quotients of \(\R[x]\).

Let \(f(x)\in \R[x]\) be any irreducible quadratic polynomial. Prove that \(\R[x]/(f)\cong \C\text{.}\)
Solution.
Since \(f\) is irreducible, the quotient \(K=\R[x]/(f)\) is a field. Since \(\deg f=2\text{,}\) we have \(K=\{r+s\overline{x}\colon r,s\in\R\}\text{.}\) Let \(z_0\in \C\) be a root of \(f(x)\text{.}\) Using TheoremÂ 1.4.7, the map \(\phi\colon K\rightarrow \C\) defined as \(\phi(r+s\overline{x})=r+sz_0\) is a ring homomorphism. Furthermore, since \(K\) is a field, the map \(\phi\) is injective, and thus we have \(K\cong \phi(K)\leq \C\text{.}\) We claim \(\phi(K)=\C\text{.}\) This is a result of the fact that \(\phi\) is a linear transformation of \(\R\)-vector spaces: i.e., for any \(r,s\in \R\) and \(\alpha=r_1+s_1\overline{x}, \beta=r_2+s_2\overline{x}\text{,}\) we have \(\phi(r\alpha+s\beta)=r\phi(\alpha)+s\phi(\beta)\text{,}\) as you can easily check. It then follows that \(K\cong \phi(K)\) as \(\R\)-vector spaces, and thus that \(\dim_\R \phi(K)=2\text{.}\) Since \(\phi(K)\leq \C\) and \(\dim_\R \phi(K)=\dim_\R \C=2\text{,}\) we conclude that \(\phi(K)=\C\text{.}\)
