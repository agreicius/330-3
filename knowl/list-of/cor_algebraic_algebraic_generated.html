Corollary 1.6.11. Algebraically generated is algebraic.

Let \(K/F\) be a field extension. The following statements are equivalent.
  1. \(K/F\) is algebraic.
  2. \(K=F(A)\text{,}\) where \(A\) is a (possibly infinite) set of algebraic elements.
In plain English: the field \(K\) is algebraic over \(F\) if and only if \(K\) is algebraically generated over \(F\text{.}\)

Proof.

The implication (1)\(\implies\)(2) is clear, since if \(K/F\) is algebraic, then we have \(K=F(K)\) and all elements of \(K\) are algebraic over \(F\text{.}\)
We now prove the implication (2)\(\implies\) (1). Assume \(K=F(A)\text{,}\) where \(A\) is a set of algebraic elements of \(K\text{.}\) Since by Theorem 1.6.8 we have
\begin{equation*} K=\bigcup_{\substack{A'\subseteq A\\ A' \text{ finite}}}F(A')\text{,} \end{equation*}
given any \(\beta\in K\text{,}\) we have
\begin{equation*} \beta\in F(A')=F(\alpha_1,\alpha_2,\dots, \alpha_m) \end{equation*}
for some \(A'=\{\alpha_1,\alpha_2,\dots, \alpha_m\}\subseteq A\text{.}\) Since each \(\alpha_i\) is algebraic, \(F(A')\) is finite over \(F\text{,}\) hence algebraic over \(F\) by Theorem 1.6.9. We conclude that \(\beta\) is algebraic over \(F\text{,}\) as desired.
