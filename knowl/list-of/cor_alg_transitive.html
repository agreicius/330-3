Corollary 1.6.13. Algebraic towers.

Given a tower of fields \(L/K/F\text{,}\) we have \(L/F\) algebraic if and only if \(L/K\) is algebraic and \(K/F\) is algebraic. Using logical shorthand:
\begin{equation} L/F \text{ algebraic} \iff L/K \text{ algebraic and } K/F \text{ algebraic}\text{.}\tag{1.8} \end{equation}

Proof.
