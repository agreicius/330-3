Theorem 1.19.6. Cyclotomic extensions of \(\Q\).

Let \(n\) be a positive integer. For any integer \(j\in \Z\text{,}\) we will write \(\overline{j}\) for its image \(j+n\Z\) in the quotient ring \(\Z/n\Z\text{.}\) Let \(\zeta\in \Q\) be a primitive \(n\)-th root of unity.
  1. \(\Q(\zeta)/\Q\) is a Galois extension of degree \(\phi(n)\text{.}\)
  2. Let \(G=\Gal(\Q(\zeta)/\Q)\text{.}\) For any \(\sigma\in G\text{,}\) we have \(\sigma(\zeta)=\zeta^{j}\) for some integer \(j\) with \((j,n)=1\text{.}\)
  3. Let \(G=\Gal(\Q(\zeta)/\Q)\text{.}\) The map \(\phi\colon G\rightarrow (\Z/n\Z)^*\) defined as
    \begin{equation*} \phi(\sigma)=\overline{j}=j+n\Z\text{,} \end{equation*}
    where \(\sigma(\zeta)=\zeta^j\text{,}\) is a group isomorphism. In particular, we have \(G\cong (\Z/n\Z)^*\text{.}\)

Proof.

  1. The paragraph preceding the the theorem statement proves that \(\Q(\zeta)/\Q\) is a Galois extension of degree \(\phi(n)\text{.}\)
  2. Given \(\sigma\in G\text{,}\) since \(\sigma\) is an automorphism it is easy to see that
    \begin{equation*} \ord \sigma(\zeta)=\ord \zeta=n\text{.} \end{equation*}
    It follows that we have \(\sigma(\zeta)=\zeta^j\text{,}\) for some integer \(j\text{,}\) where \((j,n)=1\text{.}\)
  3. Since \(\zeta^j=\zeta^{j'}\) if and only if \(j\equiv j' \pmod n\) if and only if \(\overline{j}=\overline{j'}\text{,}\) the map \(\phi\colon G\rightarrow (\Z/n\Z)^*\) is well defined.
    Suppose \(\sigma,\tau\in G\) satisfy
    \begin{equation} \sigma(\zeta)=\zeta^j, \tau(\zeta)=\zeta^{k}\text{.}\tag{1.35} \end{equation}
    We have
    \begin{align*} \sigma=\tau \amp \iff \sigma(\zeta)=\tau(\zeta)\\ \amp \iff \zeta^j=\zeta^{k} \\ \amp \iff j\equiv k \pmod n\\ \amp \iff \overline{j}=\overline{k}\\ \amp \iff \phi(\sigma)=\phi(\tau)\text{.} \end{align*}
    This proves that \(\phi\) is injective. Similarly, given \(\sigma\) and \(\tau\) satisfying (1.35), we have
    \begin{equation*} \sigma\tau(\zeta)=\sigma(\tau(\zeta))=\sigma(\zeta^k)=(\zeta^{k})^j=\zeta^{jk}\text{,} \end{equation*}
    in which case
    \begin{align*} \phi(\sigma\tau) \amp = \overline{jk}\\ \amp = \overline{j}\ \overline{k}\\ \amp = \phi(\sigma)\phi(\tau)\text{.} \end{align*}
    Thus \(\phi\) is a group homomorphism. Lastly, since \(\phi\) is injective, and since \(\abs{G}=\abs{(\Z/n\Z)^*}=\phi(n)\text{,}\) we conclude that \(\phi\) is also surjective, and hence a group isomorphism.
