Corollary 1.12.12. Separable irreducible in characteristic \(p\).

Let \(F\) be a field of prime characteristic \(p\text{.}\) Assume further that the Frobenius morphism \(\phi\colon F\rightarrow F\) is surjective: i.e., for all \(b\in F\text{,}\) there is an \(a\in F\) such that \(a^p=b\text{.}\)
If \(f\in F[x]\) is irreducible, then \(f\) is separable.

Proof.

Let \(f\in F[x]\) be irreducible and suppose by contradiction that \(f\) is inseparable. By TheoremÂ 1.12.10, we must have \(f'=0\text{.}\) Since \(\character F=p\text{,}\) for a positive integer \(k\) and \(a\in F\text{,}\) we have \(k\cdot a=0\) if and only if \(a=0\) or \(p\mid k\text{.}\) It follows that in order for \(f'=0\text{,}\) we must have
\begin{equation*} f(x)=b_{m}x^{mp}+b_{m-1}x^{(m-1)p}+\cdots+b_1x^p+b_0\text{.} \end{equation*}
Since the Frobenius map is surjective, there are elements \(a_j\in F\) satisfying \(a_j^p=b_j\) for all \(1\leq j\leq m\text{.}\) But then we have
\begin{align*} (a_{m}x^{(m-1)p}+a_{m-1}x^{(m-1)p}+\cdots+a_1x^p+a_0)^p\amp =a_{m}^p(x^{(m-1)p})^p+\cdots+a_1^px^p+a_0^p \amp (\text{ first-year's dream}) \\ \amp = b_{m}x^{mp}+b_{m-1}x^{(m-1)p}+\cdots+b_1x^p+b_0\text{,} \end{align*}
in which case
\begin{equation*} f(x)=(a_{m}x^{(m-1)p}+a_{m-1}x^{(m-1)p}+\cdots+a_1x^p+a_0)^p \end{equation*}
is not irreducible. Contradiction!
