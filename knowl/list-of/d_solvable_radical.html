Definition 1.22.6. Solvable by radicals.

Let \(F\) be a field. An element \(\alpha\in \overline{F}\) is solvable by radicals over \(F\) if \(\alpha\in L\) for some root extension \(L/F\text{.}\) A polynomial \(f(x)\in F[x]\) is solvable by radicals if all roots of \(f\) are solvable by radicals over \(F\text{.}\) An extension \(K/F\) is solvable by radicals over \(F\) if we have \(K\leq L\) for some root extension \(L/F\text{.}\)
