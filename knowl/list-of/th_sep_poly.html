Theorem 1.12.7. Separability and derivative.

Let \(F\) be a field. A nonconstant polynomial \(f\) is separable if and only if \(f\) and \(f'\) are relatively prime. Using logical shorthand:
\begin{equation*} f \text{ separable} \iff (f,f')=1\text{.} \end{equation*}

Proof.

We prove each implication separately. Let \(\overline{F}/F\) be an algebraic closure of \(F\text{.}\)
For the forward direction, assume \(f\) is separable. We will show that \(f\) and \(f'\) have no common linear factor \((x-\alpha)\) over \(\overline{F}\text{,}\) from whence it follows that they have no common nonconstant factor over \(F\text{.}\) To this end, suppose we have \(f(x)=(x-\alpha)g(x)\text{.}\) Since \(f\) is separable, the root \(\alpha\) is simple, and hence \((x-\alpha)\nmid g(x)\text{.}\) But then we have
\begin{equation*} f'(x)=(x-\alpha)'g(x)+(x-\alpha)g'(x)=g(x)+(x-\alpha)g'(x), \end{equation*}
in which case \(f'(\alpha)=g(\alpha)+0=g(\alpha)\ne 0\text{,}\) since \(\alpha\) is not a root of \(g\text{.}\) Thus \(f\) and \(f'\) have no common roots, and hence no common linear factor, as desired.
For the reverse implication, assume \(f\) and \(f'\) are relatively prime. Recall that this is equivalent to \((f,f')=(1)\) as ideals in \(F[x]\text{,}\) in which case we have \(1=g(x)f(x)+h(x)f'(x)\) for some \(g,h\in F[x]\text{.}\) Suppose by contradiction that \(f\) is not separable, and that \(\alpha\) is a repeated root. We can factor \(f\) over \(\overline{F}\) as \(f(x)=(x-\alpha)^2g(x)\) for some \(g\in K[x]\text{,}\) in which case
\begin{equation*} f'(x)=2(x-\alpha)g(x)+(x-\alpha)g'(x)=(x-\alpha)(2g(x)+g'(x)). \end{equation*}
Thus \(\alpha\) is a common root of \(f\) and \(f'\) over \(\overline{K}\text{.}\) But this is a contradiction since then weâ€™d have
\begin{equation*} 1=g(\alpha)f(\alpha)+h(\alpha)f'(\alpha)=0+0=0\text{.} \end{equation*}
We conclude that \(f\) is separable, as desired.
