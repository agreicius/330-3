Theorem 1.22.4. Cyclic extensions.

Fix a positive integer \(n\text{.}\) Assume \(F\) is a field of characteristic not dividing \(n\text{,}\) and that \(F\) contains a primitive \(n\)-th root of unity.
  1. Given any \(a\in F\text{,}\) the simple radical extension \(F(\sqrt[n]{a})/F\) is cyclic and \([F(\sqrt[n]{a})\colon F]\mid n\text{.}\)
  2. If \(K/F\) is a degree-\(n\) cyclic extension, then there is an element \(a\in F\) such that \(K=F(\sqrt[n]{a})\text{.}\)

Proof.

  1. Let \(K=F(\alpha)\text{,}\) where \(\alpha=\sqrt[n]{a}\) is an \(n\)-th root of an element \(a\in F\text{.}\) We must show that \(K/F\) is Galois, and that its Galois group is cyclic. If \(a=0\text{,}\) then \(K=F\text{,}\) in which case both claims are obvious. Assume henceforth that \(a\ne 0\text{.}\)
    Let \(f(x)=x^n-a\text{.}\) We claim that \(K=K_f\text{:}\) i.e., \(K\) is the splitting field of \(x^n-a\text{.}\) Since clearly \(K\leq K_f\text{,}\) it suffices to show (by minimality) that \(f\) splits completely in \(K\text{.}\) But if \(\beta\) is a root of \(f\text{,}\) then \(\alpha^n=\beta^n=a\text{,}\) in which case \((\alpha/\beta)^n=1\text{.}\) Thus \(\alpha/\beta\in \mu_n=\angvec{\zeta}\text{,}\) showing that \(\alpha/\beta=\zeta^k\) for some \(k\text{,}\) and hence \(\beta=\alpha\zeta^{-k}\in K\text{,}\) since \(\zeta\in F\text{.}\) Thus \(f\) splits completely in \(K\text{,}\) as desired.
    To see that \(K/F\) is separable, note that \(f'(x)=nx^{n-1}\text{.}\) Since \(\character F\nmid n\text{,}\) \(f'(x)\) is a nonzero polynomial whose only irreducible factor is \(x\text{.}\) Since \(x\) is not a factor \(f(x)\text{,}\) we see that \((f,f')=1\text{,}\) and hence that \(f\) is a separable polynomial. Since \(K=K_f\) is generated over \(F\) by the roots of \(f\text{,}\) we conclude that \(K/F\) is separable.
    Lastly, we show that \(G=\Gal(K/F)\) is cyclic. The usual argument shows that for any \(\sigma\in G\text{,}\) we must have
    \begin{equation*} \alpha\xmapsto{\ \sigma \ } \alpha \zeta_\sigma \end{equation*}
    for a unique \(n\)-th root of unity \(\zeta_\sigma\in \mu_n=\angvec{\zeta}\text{,}\) and further more that \(\sigma\) is uniquely determined by this value \(\sigma(\alpha)=\zeta_\sigma\text{.}\) It follows that the map
    \begin{align*} \phi\colon G \amp \rightarrow \mu_n\\ \sigma \amp \mapsto \zeta_\sigma \end{align*}
    is injective. It is not difficult to show, using the usual arguments, that \(\phi\) is a group homomorphism. Thus \(G\) is isomorphic to its image \(\phi(G)\leq \mu_n\text{.}\) Since \(\mu_n\) is cyclic of cardinality \(n\text{,}\) and since all subgroups of cyclic groups are cyclic, we conclude that \(G\cong \phi(G)\) is cyclic. Furthermore since \(\phi(G)\) is a subgroup of \(\mu_n\text{,}\) we have \(\abs{G}\mid n\text{,}\) and thus \([K\colon F]=\abs{G}\) divides \(n\text{.}\)
