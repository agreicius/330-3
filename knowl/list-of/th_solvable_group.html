Theorem 1.22.8. Solvable groups.

We gather some general facts about solvable groups. In what follows all groups are assumed to be finite.
  1. If \(G\) is solvable, then all subgroups of \(G\) are solvable.
  2. Let \(N\) be a normal subgroup of \(G\text{.}\) We have \(G\) solvable if and only if \(N\) and \(G/N\) are normal. In particular, quotients of solvable groups are solvable.
  3. If \(G\cong \prod_{i=1}^n G_i \text{,}\) then \(G\) is solvable if and only if \(G_i\) is solvable for all \(1\leq i\leq n\text{.}\)
  4. For all \(n\geq 5\) the alternating subgroup \(A_n\) is simple and nonabelian. As a result, \(A_n\) and \(S_n\) are solvable if and only if \(n\leq 4\text{.}\)
