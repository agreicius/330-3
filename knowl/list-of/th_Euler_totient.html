Theorem 0.4.5.

The Euler totient function \(\phi\) satisfies the following properties.
  1. If \(\gcd(a,b)=1\text{,}\) then \(\phi(ab)=\phi(a)\phi(b)\text{.}\)
  2. If \(p\) is a prime integer, then
    \begin{align*} \phi(p) \amp = p-1\\ \phi(p^{n}) \amp = p^n-p^{n-1}=p^{n-1}(p-1)\text{.} \end{align*}
