Example 1.5.4. Simple extension with algebraic generator.

Let \(K=F(\alpha)\text{,}\) where \(\alpha\) is algebraic. Prove that \(K\) is algebraic over \(F\text{.}\)
Solution.
According to Theorem 1.4.14, we have \(\dim_F K=[K:F]=n\) for some finite \(n\text{.}\) It follows that given any \(\beta\in K\text{,}\) the set \(\{\beta^n,\beta^{n-1},\dots, \beta, 1\}\) is linearly dependent, since it has more than \(n\) elements (1.3.3,“street smarts”). Thus there is a nontrivial linear combination
\begin{equation*} a_n\beta^n+a_{n-1}\beta^{n-1}+\cdots +a_1\beta+a_0=0\text{.} \end{equation*}
In other words, we have \(f(\beta)=0\text{,}\) where \(f(x)=\anpoly\text{.}\) This proves \(\beta\) is algebraic, as desired.
