Theorem 1.7.6. Splitting field generated by roots.

Let \(F\) be a field, let \(\mathcal{P}\subseteq F[x]\) a set of polynomials, and suppose \(K/F\) is a field extension that splits \(\mathcal{P}\text{.}\) Define
\begin{equation*} A=\{\alpha\in K\colon f(\alpha)=0 \text{ for some } f\in \mathcal{P}\}\text{:} \end{equation*}
i.e., \(A\) is the set of all the roots of all the polynomials in \(\mathcal{P}\text{.}\) The field \(F(A)\) generated over \(F\) by all the roots of the polynomials in \(\mathcal{P}\) is a splitting field of \(\mathcal{P}\text{.}\)

Proof.

It is clear that \(F(A)\) splits \(\mathcal{P}\text{,}\) since \(A\) contains all the roots of all the polynomials of \(\mathcal{P}\text{.}\) We must show that \(F(A)/F\) contains no proper subextension \(K'/F\) that splits \(\mathcal{P}\text{.}\) This is clear: if \(K'/F\) is a subextension of \(F(A)/F\)that splits \(\mathcal{P}\text{,}\) then \(K'\) must contain all roots of all polynomials of \(\mathcal{P}\text{;}\) that is, \(K'\) must contain \(A\text{.}\) Since by definition \(F(A)\) is the smallest extension of \(K/F\) containing \(F\) and \(A\text{,}\) we have \(F(A)\leq K'\text{.}\) Since \(K'\leq F(A)\text{,}\) we conclude \(K'=F(A)\text{.}\)
