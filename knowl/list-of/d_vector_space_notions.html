Definition 1.2.8. Vector space notions.

Let \(V\) be an \(F\)-vector space.
  • Subspace.
    A subset \(W\subseteq V\) is a subspace of \(V\text{,}\) denoted \(W\leq V\text{,}\) if it is an \(F\)-submodule: i.e., it is closed under addition and scalar multiplication.
  • Span.
    Given a set \(S\subseteq V\) the span of \(S\text{,}\) denoted \(\Span S\)is the set of all \(F\)-linear combinations of elements of \(S\text{:}\) i.e.,
    \begin{equation*} \Span S=\{ \boldv\in V\colon \boldv=a_1\boldv_1+a_2\boldv_2+\cdots +a_n\boldv_n \text{ for some } n\in \Z_+, a_i\in F, \boldv_i\in S\}\text{.} \end{equation*}
    A set \(S\) spans the vector space \(V\) if \(\Span S=V\text{.}\)
  • Linear independence.
    A set \(S\subseteq V\) is linearly independent if for any distinct elements \(\boldv_1,\boldv_2,\dots, \boldv_n\in S\) and scalars \(a_1,a_2,\dots, a_n\in F\text{,}\) if \(a_1\boldv_1+a_2\boldv_2+\cdots+a_n\boldv_n=\boldzero\text{,}\) then \(a_i=0\) for all \(1\leq i\leq n\text{.}\)
    A set \(S\) is linearly dependent if and only if it is not linearly independent.
  • Basis.
    A set \(B\subseteq V\) is a basis of \(V\) if it spans \(V\) and is linearly independent.
  • Dimension.
    The space \(V\) is finite dimensional if it contains a finite basis \(B\text{.}\) If \(\abs{B}=n\text{,}\) then given any other basis \(B'\) of \(V\text{,}\) we have \(\abs{B'}=n\text{.}\) We define the dimension of \(V\) to be \(n\text{,}\) denoted \(\dim_F V=n\text{.}\)
    The space \(V\) is infinite dimensional if it does not contain a finite basis. We write \(\dim_F V=\infty\) in this case.
