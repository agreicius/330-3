    Section 1.14 Separable finite extensions

    In this section we re-cast the notion of separability in terms of extensions of embeddings. The two concepts seem quite removed from one another until you consider the special case of a simple algebraic extension \(F(\alpha)/F\text{.}\) Let \(f(x)=m_{\alpha,F}(x)\text{,}\) and let \(n=\deg f\text{.}\) As we saw in Theorem 1.10.3, given an embedding \(\sigma\colon F\rightarrow L\text{,}\) the set of extensions of \(\sigma\) to \(F(\alpha)\) is in bijective correspondence with the set of roots of \(f^\sigma(x)\in L[x]\text{.}\) The number of these roots is at most \(n\) (since \(\deg f^\sigma=\deg f=n\)), and is equal to \(n\) exactly when \(f\) (and \(f^\sigma\)) have \(n\) distinct roots: i.e., when \(f\) is separable. This is the crucial idea behind the proof of Theorem 1.14.1, which gives us a characterization of separability in terms of extensions of embeddings. This equivalent formulation, abstract as it might sound, is very useful as a means of verifying that a field extension is separable.

    Theorem 1.14.1. Separable finite extensions.

    Let \(K/F\) be a finite field extension of degree \(n=[K\colon F]\text{.}\) The following statements are equivalent.
    1. Separable.
      \(K/F\) is separable.
    2. Separably generated.
      \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\) where each \(\alpha_i\) is separable over \(F\text{.}\)
    3. Maximal separable degree.
      For any embedding \(\sigma\colon F\rightarrow L\text{,}\) with \(L\) algebraically closed, there are exactly \(n=[K\colon F]\) extensions of \(\sigma\) to \(K\text{.}\)

    Proof.

    We prove the cycle of implications (1)\(\implies\)(2)\(\implies\)(3)\(\implies\)(1).

    Implication:(1)\(\implies\)(2).

    Assume \(K/F\) is finite. We have \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\) for some elements \(\alpha_i\) by Theorem 1.6.9. Since \(K/F\) is separable, each \(\alpha_i\) is separable over \(F\text{.}\)

    Implication:(2)\(\implies\)(3).

    Assume \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\) with each \(\alpha_i\) separable over \(F\text{.}\) We show by induction on \(n\) that \(K/F\) satisfies the condition described in (3).
    Base case: \(n=1\text{.}\) In this case we have \(K=F(\alpha)\text{.}\) Since \(\alpha\) is separable over \(F\text{,}\) its minimal polynomial \(f(x)=m_{\alpha,F}(x)\) is separable. Let \(d=\deg f\text{.}\) Given an embedding \(\sigma\colon F\rightarrow L\text{,}\) where \(L\) is algebraically closed, we know by Theorem 1.10.3 that the set \(X\) of all extensions of \(\sigma\) to \(K=F(\alpha)\) is in bijective correspondence with the set \(R\) of all roots in \(L\) of the polynomial \(f^\sigma(x)\text{.}\) Since \(f\) is separable, so is \(f^\sigma\text{.}\) (See proof below.) Since \(\deg f=\deg f^\sigma\text{,}\) and since \(f^\sigma\) splits completely over the algebraically closed field \(L\text{,}\) there are \(d\) distinct roots of \(f^\sigma\text{.}\) We conclude that
    \begin{equation*} \abs{X}=\abs{R}=d=[F(\alpha)\colon F]\text{,} \end{equation*}
    as desired.
    To see why \(f^\sigma\) is separable, let \(\beta_1,\beta_2,\dots,\beta_d\) be the distinct roots of \(f\) lying in an algebraic closure \(\overline{F}\text{.}\) We can extend \(\sigma\) to an embedding \(\sigma^*\colon \overline{F}\rightarrow L\) by Theorem 1.11.1, and it is easy to see that \(\sigma^*(\beta_i)\) are the distinct roots of \(f^\sigma\text{,}\) which also has degree \(d\text{.}\)
    Induction case: let \(n> 1\) and assume the result holds for any field extension separably generated by \(n-1\) elements. Given \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\text{,}\) we have \(K=K'(\alpha_n)\text{,}\) where \(K'=F(\alpha_1,\dots, \alpha_{n-1})\text{.}\) Let \([K\colon K']=e\) and \([K'\colon F]=d\text{,}\) so that
    \begin{equation*} n=[K\colon F]=[K\colon K']\, [K'\colon F]=e\, d\text{.} \end{equation*}
    By the induction hypothesis, given any \(\sigma\colon F\rightarrow L\) with \(L\) algebraically closed, there are exactly \(d=[K'\colon F]\) extensions \(\tau'\colon K'\rightarrow L\) of \(\sigma\) to \(K'\text{.}\) Applying the base case (\(n=1\)) to the simple extension \(K=K'(\alpha_n)\text{,}\) for each such \(\tau'\colon K'\rightarrow L\) extending \(\sigma\) there are exactly \(e=[K'(\alpha)\colon K']\) extensions of \(\tau'\) to an embedding \(\tau\colon K=K'(\alpha)\colon L\text{.}\) We now use our counting extensions of embeddings lemma. Let \(X\) be the set of all extensions of \(\sigma\) to \(K\text{,}\) let \(X'\) be the set of all extensions of \(\sigma\) to \(K'\text{,}\) and for each \(\tau'\in X'\text{,}\) let \(X_{\tau'}\) be the set of all extensions of \(\tau'\) to \(K\text{.}\) (See Figure 1.10.10 for a visual summary of this setup.) Using Lemma 1.10.9, we have
    \begin{align*} \abs{X} \amp =\sum_{\tau'\in X'}\abs{X_{\tau'}}\\ \amp =\sum_{\tau'\in X'}e \amp (\text{base step})\\ \amp = e\abs{X'}\\ \amp = ed \amp (\text{induction step})\\ \amp = n \amp (\text{tower law})\text{,} \end{align*}
    as desired.

    Implication:(3)\(\implies\)(1).

    Assume \(K/F\) is finite and satisfies the condition described in (3). Let \(\overline{F}/F\) be an algebraic closure of \(F\text{.}\) Fix an \(\alpha\in K\text{,}\) and let \(f(x)=m_{\alpha,F}(x)\text{.}\) Assume \(\deg f=d\text{.}\) We wish to show that \(f\) is separable, or equivalently, that the set \(R=\{\beta\in \overline{F}\colon f(\beta)\}\) of roots of \(f\) has exactly \(d\) elements: i.e., \(\abs{R}=d\text{.}\) To this end, we apply condition (3) to the field embedding \(i\colon F\rightarrow \overline{F}\text{.}\) Namely, the set of extensions \(X\) of \(i\colon F\rightarrow \overline{F}\) to an embedding \(\tau\colon K\rightarrow \overline{F}\) has cardinality \(\abs{X}=n\text{.}\) On the other hand, we have the tower of extensions \(K/K'/F\text{,}\) where \(K'=F(\alpha)\text{,}\) and applying the counting extensions of embeddings lemma, we see that
    \begin{align*} n=\abs{X}\amp=\sum_{\tau'\in X'}X_{\tau'} \text{,} \end{align*}
    where \(X'\) is the set of all extensions of \(i\colon F\rightarrow \overline{F}\) to \(K'=F(\alpha)\text{,}\) and for each \(\tau'\in X'\text{,}\) \(X_{\tau'}\) is the set of all extensions of \(\tau'\) to \(K\text{.}\) (Again, see Figure 1.10.10 for a visual summary of this setup.) Since \(K'=F(\alpha)\) is simple, we have \(\abs{X'}\leq \abs{R}\text{,}\) the number of distinct roots of \(f\) in \(\overline{R}\text{,}\) by Theorem 1.10.3. Furthermore, for each \(\tau'\in X'\text{,}\) we have \(\abs{X_{\tau'}}\leq [K\colon K']=e\) by Theorem 1.10.11. Thus we have
    \begin{align*} n=\abs{X}\amp=\sum_{\tau'\in X'}X_{\tau'} \\ \amp \leq\sum_{tau'\in X'}e \\ \amp =e\abs{X'}\\ \amp \leq e\,d\\ \amp = n\text{.} \end{align*}
    It follows that the two inequalities above are in fact equalities, and thus in particular, that \(d=\abs{X'}=\abs{R}\text{.}\) Thus \(f\) is separable, as desired.

    Remark 1.14.2. Maximal separable degree.

    Given a finite field extension \(K/F\) of degree \([K\colon F]=n\text{,}\) its separable degree is defined as the the number of distinct extensions to \(K\) of any embedding \(\sigma\colon F\rightarrow L\text{,}\) where \(L\) is algebraically closed, and is denoted \([K\colon F]^s\text{.}\) (You can show that this number is independent of the choice of algebraically closed field \(L\) and embedding \(\sigma\colon F\rightarrow L\text{.}\)) By Theorem 1.10.11 this number \(\)of extensions is at most \(n=[K\colon F]\text{.}\) According to Theorem 1.14.1, the extension \(K/F\) is separable if and only if the number of extensions is equal to \(n=[K\colon F]\text{.}\) Thus \(K/F\) is separable if and only if it has “maximal separable degree” \(n=[K\colon F]\text{.}\)

    Corollary 1.14.3. Separable is separably generated.

    A field extension \(K/F\) is separable if and only if there is a set \(A\subseteq K\) of separable elements such that \(K=F(A)\text{.}\) In plain English: an extension is separable if and only if it is separably generated.

    Proof.

    The forward implication is easy: if \(K/F\) is separable, then all elements of \(K\) are separable over \(F\text{,}\) and we have \(K=F(A)\text{,}\) where \(A=K\text{.}\)
    Let’s prove the reverse implication. Assume \(A\) is a collection of separable elements of \(K\) that generate \(K\) over \(F\text{:}\) i.e., such that \(K=F(A)\text{.}\) Recall that we have
    \begin{equation*} K=F(A)=\bigcup_{\substack{A'\subseteq A\\ A' \text{ finit}}}F(A') \end{equation*}
    by Theorem 1.6.8. Thus, given \(\alpha\in K\text{,}\) we have
    \begin{equation*} \alpha\in F(A')=F(\alpha_1,\alpha_2,\dots, \alpha_n) \end{equation*}
    for some finite set \(A'=\{\alpha_1,\alpha_2,\dots, \alpha_n\}\subseteq A\text{.}\) Since by assumption \(A\) consists of separable elements, the extension \(F(A')/F\) is finitely separably generated over \(F\text{,}\) and hence separable by Theorem 1.14.1. It follows that \(\alpha\) is separable over \(F\text{,}\) as desired.

    Corollary 1.14.4. Separability is strongly transitive.

    Let \(L/K/F\) be a tower of algebraic field extensions. The extension \(L/F\) is separable if and only if \(L/K\) and \(K/F\) are both separable. Using logical shorthand:
    \begin{equation*} L/F \text{ separable} \iff L/K \text{ separable and } K/F \text{ separable}\text{.} \end{equation*}

    Proof.

    We prove both implications separately.

    Implication: \(\implies\).

    Assume \(L/F\) is separable. Clearly then \(K/F\) is separable, since if \(\alpha\in K\text{,}\) then \(\alpha\in L\text{,}\) and hence satisfies a separable polynomial over \(F\) since \(L/F\) is separable.
    Similarly, to see that \(L/K\) is separable, observe that given \(\alpha\in L\text{,}\) its minimal polynomial \(m_{\alpha,F}(x)\in F[x]\) is separable. Since \(F\leq K\text{,}\) we have \(m_{\alpha, F}(x)\in K[x]\text{.}\) Thus \(\alpha\) satisfies a separable polynomial over \(K\text{.}\)

    Implication: \(\impliedby\).

    Assume \(L/F\) and \(K/F\) are both separable. Given \(\alpha\in L\text{,}\) its minimal polynomial \(m_{\alpha,K}(x)=x^n+\alpha_{n-1}x^{n-1}+\cdots +\alpha_1 x+\alpha_0\) over \(K\) is separable. Let
    \begin{equation*} K'=F(\alpha_0,\alpha_1,\dots, \alpha_{n-1}). \end{equation*}
    Since each \(\alpha_i\in K\) is separable over \(F\text{,}\) \(K'\) is separable (and finite) over \(F\) by Theorem 1.14.1. Furthermore, \(K'(\alpha)/K'\) is algebraic since \(\alpha\) satisfies the separable polynomial \(m_{\alpha,K}(x)\in K'[x]\leq K[x]\text{.}\) Let \([K'(\alpha)\colon K']=e\) and \([K'\colon F]=d\) so that
    \begin{equation*} [K'(\alpha)\colon F]=[K'(\alpha)\colon K'][K'\colon F]=e\,d\text{.} \end{equation*}
    We will use condition (3) of Theorem 1.14.1 to prove \(K'(\alpha)/F\) is separable, which then implies \(\alpha\) is separable over \(F\text{.}\) Given an embedding \(\sigma\colon F\rightarrow L\text{,}\) where \(L\) is algebraically closed, since \(K'/F\) is separable, the set \(X'\) of extensions of \(\sigma\) to \(K'\) has cardinality \(\abs{X'}=[K'\colon F]=d\) by Theorem 1.14.1. Similarly, since \(K'(\alpha)/K'\) is separable, for each \(\tau'\in X'\text{,}\) the set \(X_{\tau'}\) of extensions of \(\tau'\) to \(K'(\alpha)\) has cardinality \(\abs{X_{\tau'}}=[K'(\alpha)\colon K']=e\text{.}\) By Lemma 1.10.9, it follows that there are
    \begin{equation*} ed=[K'(\alpha)\colon F] \end{equation*}
    extensions of \(\sigma\colon F\rightarrow L\) to \(K'(\alpha)\text{.}\) We conclude from Theorem 1.14.1 that \(K'(\alpha)/F\) is separable, as desired.

    Theorem 1.14.5. Primitive element theorem.

    Let \(K/F\) be a finite separable extension. There is an element \(\alpha\in K\) such that \(K=F(\alpha)\text{.}\) Using logical shorthand:
    \begin{equation*} K/F \text{ finite separable} \implies K=F(\alpha) \text{ for some } \alpha\in K\text{.} \end{equation*}

    Proof.

    First consider the case where \(F\text{,}\) and hence also \(K\text{,}\) is a finite field. Since \(K\) is finite, we have \(K^*=\langle \alpha\rangle=\{1,\alpha,\alpha^2,\dots, \alpha^r\}\) for some \(\alpha\text{,}\) in which case clearly \(K=F(\alpha)\text{,}\) as desired.
    From now on we assume \(F\) is infinite. Since \(K/F\) is finite, we can write \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\text{.}\) We will show by induction on \(n\) that if \(K\) is separable, then \(K=F(\alpha)\) for some \(\alpha\in K\text{.}\) The base case \(n=1\) is trivial.
    For the induction argument, given a separable extension \(K=F(\alpha_1,\alpha_2,\dots, \alpha_n)\text{,}\) by induction \(F(\alpha_1,\dots, \alpha_{n-1})=F(\alpha)\) for some \(\alpha\text{,}\) in which case \(K=F(\alpha, \alpha_n)\text{.}\) Thus, to complete the proof it suffices to show that if \(K=F(\alpha,\beta)\) is separable, then we have \(K=F(\gamma)\) for some \(\gamma\in K\text{.}\) In fact, we will show that we can pick \(\gamma=\alpha+c\beta\) for some \(c\in F\text{,}\) and this is precisely where we will use the fact that \(F\) is infinite. In more detail, we will show that the set of \(c\in F\) such that \(F(\alpha+c\beta)\ne K\) is finite, and thus, since \(F\) is infinite, there is a \(c\in F\) such that \(F(\alpha+c\beta)=K\text{.}\)
    Here is where we use the fact that \(K=F(\alpha,\beta)\) is separable over \(F\text{.}\) Let \([K\colon F]=n\text{,}\) and let \(X=\{\sigma_1,\sigma_2,\dots, \sigma_n\}\) be the \(n\) extensions of the inclusion map \(i\colon F\hookrightarrow \overline{F}\text{,}\) for some algebraic closure \(\overline{F}/F\text{.}\) If \(K'=F(\alpha+c\beta)\ne K\text{,}\) then we have \([K\colon K']\text{.}\) Let \(\sigma\colon K'\rightarrow \overline{F}\) be any extension of the inclusion map \(i\colon F\hookrightarrow \overline{F}\text{.}\) By strong transitivity, \(K/K'\) is separable; and since \([K\colon K']> 1\text{,}\) there are at least two distinct extensions \(\tau_i\ne \tau_j\in X\) of \(\sigma\) to \(K\text{.}\)
    Since \(\tau_i\vert_{K'}=\tau_j\vert_{K'}=\sigma\text{,}\) we have
    \begin{equation*} \tau_i(\alpha+c\beta)=\tau_j(\alpha+c\beta)\text{,} \end{equation*}
    and thus
    \begin{equation*} \tau_i(\alpha)+c\tau_i(\beta)=\tau_j(\alpha)+c\tau_j(\beta)\text{.} \end{equation*}
    If \(\tau_i(\beta)=\tau_j(\beta)\text{,}\) then we’d also have \(\tau_i(\alpha)=\tau_j(\alpha)\text{,}\) in which case \(\tau_i=\tau_j\text{,}\) since \(K=F(\alpha,\beta)\text{.}\) This is a contradiction, as we assumed \(\tau_i\ne \tau_j\text{.}\) We conclude that \(\tau_i(\beta)\ne \tau_j(\beta)\text{,}\) in which case we can solve for \(c\) as
    \begin{equation*} c=\frac{\tau_i(\alpha)-\tau_{j}(\alpha)}{\tau_i(\beta)-\tau_{j}(\beta)}\text{.} \end{equation*}
    This means that \(c\) must be an element of the finite set of elements
    \begin{equation*} \left\{\frac{\tau(\alpha)-\tau'(\alpha)}{\tau(\beta)-\tau'(\beta)}\colon \tau\ne \tau'\in X, \tau(\beta)\ne\tau'(\beta) \right\}\text{.} \end{equation*}
    It follows that there are only finitely many such \(c\in F\) for which \(F(\alpha+c\beta)\ne F(\alpha,\beta)\text{,}\) as desired.

    Remark 1.14.6. Primitive element theorem.

    Given a simple algebraic extension \(K=F(\alpha)\text{,}\) the simple generator \(\alpha\) is often called a primitive element. Thus the primitive element theorem can be summarized as follows: every finite separable extension has a primitive element.

    Example 1.14.7. Primitive element theorem.

    Let \(K=\Q(\sqrt{2},\sqrt{3})\text{.}\) Show that \(\alpha=\sqrt{2}+c\sqrt{3}\) is a primitive element of \(K\) for any nonzero \(c\in \Q\text{.}\)

    Example 1.14.8. Primitive element theorem.

    Let \(K\leq \C\) be a splitting field of \(f(x)=x^3-2\in \Q\text{.}\) Find a primitive element of \(K/\Q\text{.}\)
