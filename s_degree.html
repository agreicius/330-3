    Section 1.3 Degree of a field extension

    Subsection Vector spaces

    In general, given a ring extension \(R\leq S\text{,}\) the ring \(S\) is naturally a module over the ring \(R\text{,}\) where the module addition operation is just the ring addition of \(S\text{,}\) and where scalar multiplication of a “scalar” \(r\in R\) and “module element” \(s\in S\) is defined as the ring product \(rs\text{.}\) In the special case of a field extension \(F\leq K\text{,}\) the field \(K\) is a module over the the field \(F\text{,}\) and is thus an \(F\)-vector space.

    Definition 1.3.1. Vector space.

    Let \(F\) be a field. We call an \(F\)-module \(V\) a vector space over \(F\) (or an \(F\)-vector space).

    Definition 1.3.2. Vector space notions.

    Let \(V\) be an \(F\)-vector space.
    • Subspace.
      A subset \(W\subseteq V\) is a subspace of \(V\text{,}\) denoted \(W\leq V\text{,}\) if it is an \(F\)-submodule: i.e., it is closed under addition and scalar multiplication.
    • Span.
      Given a set \(S\subseteq V\) the span of \(S\text{,}\) denoted \(\Span S\)is the set of all \(F\)-linear combinations of elements of \(S\text{:}\) i.e.,
      \begin{equation*} \Span S=\{ \boldv\in V\colon \boldv=a_1\boldv_1+a_2\boldv_2+\cdots +a_n\boldv_n \text{ for some } n\in \Z_+, a_i\in F, \boldv_i\in S\}\text{.} \end{equation*}
      A set \(S\) spans the vector space \(V\) if \(\Span S=V\text{.}\)
    • Linear independence.
      A set \(S\subseteq V\) is linearly independent if for any distinct elements \(\boldv_1,\boldv_2,\dots, \boldv_n\in S\) and scalars \(a_1,a_2,\dots, a_n\in F\text{,}\) if \(a_1\boldv_1+a_2\boldv_2+\cdots+a_n\boldv_n=\boldzero\text{,}\) then \(a_i=0\) for all \(1\leq i\leq n\text{.}\)
      A set \(S\) is linearly dependent if and only if it is not linearly independent.
    • Basis.
      A set \(B\subseteq V\) is a basis of \(V\) if it spans \(V\) and is linearly independent.
    • Dimension.
      The space \(V\) is finite dimensional if it contains a finite basis \(B\text{.}\) If \(\abs{B}=n\text{,}\) then given any other basis \(B'\) of \(V\text{,}\) we have \(\abs{B'}=n\text{.}\) We define the dimension of \(V\) to be \(n\text{,}\) denoted \(\dim_F V=n\text{.}\)
      The space \(V\) is infinite dimensional if it does not contain a finite basis. We write \(\dim_F V=\infty\) in this case.

    Theorem 1.3.3. Dimension theory compendium.

    Let \(V\) be an \(F\)-vector space of dimension \(n\text{.}\)
    1. Contract.
      If \(S\) spans V, then it can be contracted to a basis of \(V\text{:}\) i.e., we can find a basis \(B\) with \(B\subseteq S\text{.}\)
    2. Expand.
      If \(S\) is a linearly independent set, then it can be expanded to a basis of \(V\text{:}\) i.e., we can find a basis \(B\) with \(S\subseteq B\text{.}\)
    3. Street smarts.
      Let \(S\) be a subset of \(V\text{.}\) If \(\val{S} < n\text{,}\) then \(S\) does not span \(V\text{;}\) if \(\val{S} > n\text{,}\) then \(S\) is linearly dependent; if \(\val{S}=n\text{,}\) then \(S\) spans \(V\) if and only if \(S\) is linearly independent if and only if \(S\) is a basis.
    4. Dimension of subspaces.
      If \(W\subseteq V\) is a subspace, then
      1. \(\dim W\leq \dim V\text{,}\) and
      2. \(\dim W=\dim V\) if and only if \(W=V\text{.}\)

    Procedure 1.3.4. Computing dimension.

    To compute the dimension of an \(F\)-vector space \(V\) proceed as follows.
    1. Basis.
      Attempt to produce a basis \(B\) of \(V\text{.}\)
    2. Dimension.
      • If you found a finite basis \(B\) in Step 1, then \(\dim_F V=\abs{B}\text{.}\)
      • If you found an infinite basis in Step 1, or were able to show that no finite basis exists, then \(\dim_F V=\infty\text{.}\)
        A common technique for showing that no finite basis exists is to exhibit an infinite linearly independent set.

    Subsection Degree of an extension

    Definition 1.3.5. Degree of an extension.

    The degree of a field extension \(K/F\text{,}\) denoted \([K\colon F]\text{,}\) is the dimension of \(K\) as an \(F\)-vector space: i.e.,
    \begin{equation*} [K\colon F]=\dim_F K\text{.} \end{equation*}
    An extension \(K/F\) is finite if \([K\colon F]=n< \infty\) and infinite if \([K\colon F]=\infty\text{.}\)

    Example 1.3.6. Degree of \(\C\) over \(\R\).

    Consider the extension \(\C/\R\text{.}\) Essentially by definition, the set \(B=\{1, i\}\) is a basis of \(\C\) over \(\R\text{,}\) and thus \([\C\colon \R]=\dim_\R \C=\abs{B}=2\text{.}\)

    Example 1.3.7. Degree of quadratic extension.

    Let \(K=\Q(\sqrt{d})\text{,}\) where \(d\) is a square-free positive integer. Compute \([K\colon \Q]\text{.}\)
    Solution.
    First we claim that \(K=\Q(\sqrt{d})=\{a+b\sqrt{d}\colon a,b\in \Q\}\text{.}\) This is shown in much the same manner as in Example 1.2.8. First we show that \(\{a+b\sqrt{d}\colon a,b\in \Q\}\) is indeed a field (easy), and then we follow Procedure 1.2.7.
    Next we will show that \([K\colon \Q]=\dim_\Q K=2\text{,}\) following Procedure 1.3.4. We claim that the set \(B=\{1, \sqrt{d}\}\) is basis of \(K\) over \(\Q\text{.}\) Since \(K=\{a+b\sqrt{d}\colon a,b\in \Q\}\text{,}\) we see that \(\Span B=K\text{.}\) To see that \(B\) is linearly independent, assume we have \(a+b\sqrt{d}=0\) for some \(a,b\in \Q\text{.}\) If \(a\ne 0\text{,}\) then \(\sqrt{d}=-b/a\in \Q\text{,}\) and thus \(\sqrt{d}\) is a rational root of the primitive integer polynomial \(x^2-d\text{.}\) Gauss’s lemma then implies that \(\sqrt{d}\) is an integer; but this is clearly impossible, since \((\sqrt{d})^2=d\text{,}\) and \(d\) is assumed to be a square-free integer. Thus we must have \(a=0\text{.}\) But then \(b\sqrt{d}=0\) implies \(b=0\text{,}\) since \(\sqrt{d}\ne 0\text{.}\)
    Having shown that \(B=\{1,\sqrt{d}\}\) is a basis of \(K\) over \(\Q\text{,}\) we conclude that \([K\colon \Q]=\dim_\Q K=\abs{B}=2\text{.}\)

    Example 1.3.8. Degree of rational functions.

    Let \(F\) be a field and consider the field extension \(F\leq F(x)\text{,}\) where \(F(x)\) is the field of rational functions in one variable over \(F\text{.}\) Compute \([F(x)\colon F]\text{.}\)
    Solution.
    We claim \([F(x)\colon F]=\infty\text{.}\) To prove the claim it is enough, thanks to Theorem 1.3.3, to show that there is an infinite linearly independent set. To this end, we claim that the set \(S=\{1,x,x^2\dots \}\subseteq F(x)\) of all nonnegative powers of \(x\) is linearly independent over \(F\text{.}\) Indeed, a finite linear combination of elements of $x$ is nothing more than a polynomial \(f(x)=\anpoly\text{,}\) where \(a_i\in F\) for all \(1\leq i\leq n\text{.}\) By definition of the polynomial ring \(F[x]\text{,}\) which we think of as a subset of \(F(x)\text{,}\) we have
    \begin{equation*} \anpoly=0\iff a_i=0 \text{ for all } 1\leq i\leq n\text{.} \end{equation*}
    This proves \(S\) is linearly independent, and hence that \(F(x)/F\) is an infinite extension.
